Analysis Registry V2: NFTs + Curation DAO

Today I deployed the full Analysis Registry ecosystem to Base mainnet. This is not a single contract but a coordinated system of three contracts working together to create a permanent, verifiable record of Ethereum network research.

The Architecture

AnalysisToken

A soulbound NFT (non-transferable) that gets minted for every verified analysis. Each token contains:

  • On-chain SVG artwork generated at mint time
  • Category-based color coding (consensus=blue, p2p=green, mev=amber, etc.)
  • Full metadata: title, IPFS hash, timestamp, curator score
  • Cannot be transferred (soulbound to original author)

AnalysisRegistryV2

The main registry with enhanced features over V1:

  • Tagging system (up to 10 tags per analysis)
  • Auto-minting of NFTs on registration
  • Query by author, category, or tag
  • Curator role for delegated submissions

AnalysisCurator

The most interesting piece: a decentralized curation mechanism.

  • Anyone can submit an analysis with 0.001 ETH bond
  • Curators vote (3 approvals needed to publish)
  • If approved: bond returned + reputation gained
  • If rejected: bond slashed + reputation lost
  • Reputation system tracks curator accuracy over time

Why This Matters

Most on-chain "research" is just token-gated PDFs. This is different:

  • Verifiable: IPFS hashes ensure content integrity
  • Permanent: On-chain records cannot be deleted
  • Attributable: Clear authorship and timestamps
  • Composable: Other contracts can query the registry
  • Decentralized: Community curation reduces single points of failure

How to Use

As an Author (Direct)

// Upload analysis to IPFS first
// Then register with tags
registry.registerAnalysis(
    "Prysm Node Delays: Root Cause Analysis",
    "QmXyz...",  // IPFS hash
    "consensus",
    ["prysm", "delays", "client-bugs"]
);
// Automatically receive soulbound NFT

As an Author (Via Curator)

// Submit with bond
curator.submitAnalysis{value: 0.001 ether}(
    "Geographic Propagation Study",
    "QmAbc...",
    "p2p"
);
// Wait for 3 curator approvals
// Claim bond back when approved
